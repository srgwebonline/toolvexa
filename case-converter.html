Case Type Examples

Sentence Case

Capitalizes the first letter of each sentence.

this is a sentence. this is another sentence.
This is a sentence. This is another sentence.

Title Case

Capitalizes the first letter of each word (except articles, conjunctions, prepositions).

the lord of the rings
The Lord of the Rings

camelCase

First word lowercase, subsequent words capitalized, no spaces.

user account manager
userAccountManager

PascalCase

Each word capitalized, no spaces. Also called UpperCamelCase.

customer data service
CustomerDataService

snake_case

All lowercase with underscores between words.

database table name
database_table_name

kebab-case

All lowercase with hyphens between words. Also called dash-case.

css class selector
css-class-selector

Frequently Asked Questions

What is the difference between camelCase and PascalCase?

camelCase (also called lowerCamelCase) starts with a lowercase letter and capitalizes the first letter of each subsequent word (e.g., "userName", "totalAmount"). PascalCase (also called UpperCamelCase) capitalizes the first letter of every word including the first one (e.g., "UserName", "TotalAmount"). Both remove spaces between words.

When should I use snake_case vs kebab-case?

snake_case (with underscores) is commonly used in Python, Ruby, and database field names (e.g., "user_name", "created_at"). kebab-case (with hyphens) is often used in CSS classes, URLs, and filenames (e.g., "user-name", "created-at"). Both are all lowercase with separators between words.

Does the converter handle special characters and numbers?

Yes, our converter preserves all special characters, numbers, and punctuation. Only alphabetical characters are affected by case conversion. For example, "user123@email.com" converted to uppercase becomes "USER123@EMAIL.COM" - the numbers and symbols remain unchanged.

What is Title Case and how is it different from Uppercase?

Title Case capitalizes the first letter of each major word (typically nouns, verbs, adjectives, adverbs) while keeping articles (a, an, the), conjunctions (and, but, or), and short prepositions (in, on, at) lowercase unless they start the title. UPPERCASE converts ALL letters to uppercase regardless of their position in words.

Is there a limit to how much text I can convert?

No, there's no practical limit. Since all processing happens locally in your browser, you can convert as much text as your device can handle. However, for extremely large documents (over 1MB), you might experience slight delays depending on your device's performance.

Does the tool work offline?

Yes! Once the page is loaded, all text conversion happens locally in your browser using JavaScript. No internet connection is required for the conversion functionality. You only need internet for the initial page load and for accessing external resources like fonts.
